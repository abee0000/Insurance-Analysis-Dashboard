# Insurance Analytics Dashboard 📊

An end-to-end Insurance Analytics solution developed to transform raw insurance data into actionable insights using SQL, Excel, Tableau, and Power BI. The project focuses on analyzing policy, customer, claims, and payment data to generate business KPIs and interactive dashboards for management reporting and decision-making.

The system integrates data cleaning, SQL-based KPI generation, and dashboard visualization to provide a unified analytics platform for tracking policy performance, customer engagement, claims processing, premium growth, and payment status.

## 🚀 Features
- 📈 Interactive Insurance Analytics Dashboards
- 🗄️ SQL-based KPI generation and reporting
- 📊 Excel dashboard for quick analysis
- 📉 Power BI dashboard for advanced visualization
- 📋 Tableau dashboard for business insights
- 🔍 Customer, policy, claim, and payment analysis
- 📅 Policy renewal and expiry tracking
- 💰 Premium growth and claim amount analysis

## 📌 Key Performance Indicators (KPIs)
- Total Policies
- Total Customers
- Age Bucket Wise Policy Count
- Gender Wise Policy Count
- Policy Type Wise Policy Count
- Policies Expiring This Year
- Premium Growth Rate
- Claim Status Wise Policy Count
- Payment Status Wise Policy Count
- Total Claim Amount

## 🛠️ Technologies Used
- SQL Server / MySQL
- Microsoft Excel
- Power BI
- Tableau
- SQL Queries & Joins
- Pivot Tables & Charts
- Data Cleaning & Transformation

## 📂 Project Workflow
1. Import and clean raw insurance datasets
2. Create relational database tables in SQL
3. Generate KPIs using SQL queries
4. Build interactive dashboards in:
   - Excel
   - Power BI
   - Tableau
5. Analyze business performance and trends

## 💡 Applications
- Insurance business reporting
- Claims and risk analysis
- Customer segmentation
- Revenue and premium analysis
- Policy renewal tracking
- Executive and management dashboards

## ✅ Outcome
The project successfully delivers a unified Insurance Analytics solution combining SQL, Excel, Tableau, and Power BI to generate business-ready dashboards and KPIs. The system enables better decision-making by providing clear insights into policy management, customer behavior, claims performance, and revenue growth.
